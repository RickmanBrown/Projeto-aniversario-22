# 🎂 Surpresa de Aniversário

Um projeto front-end interativo feito com **HTML, CSS e JavaScript**, criado como uma surpresa de aniversário em estilo visual inspirado em **pixel art retrô 2D**.

A experiência funciona como uma pequena cena interativa:
- o usuário clica na porta para entrar
- a sala aparece escura
- ao ligar a luz, a festa é revelada
- personagens aparecem ao redor da mesa
- Alan se aproxima com um presente
- acontece um diálogo
- o presente é entregue
- depois surge a carta final

## ✨ Ideia do projeto

Este projeto foi criado como uma experiência visual e afetiva, unindo:
- interatividade
- animação simples
- personagens em pixel art
- ambientação de festa
- narrativa em forma de cena

A proposta foi desenvolver algo com personalidade, emoção e identidade visual, fugindo de projetos genéricos e criando uma lembrança especial em forma de código.

## 🛠 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
